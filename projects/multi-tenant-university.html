Multi-tenant University Website

December 10, 2024

Tech Stack: Next.js, MongoDB, Node.js, TailwindCSS

Key Features:

  • Role-Based Authentication & Authorization: Secure login system that dynamically assigned permissions based on user type and university.
  • Multi-Tenant Design: Supported multiple universities within the same application while maintaining data isolation.
  • Dynamic Dashboards: Customized views for admins (user management, course setup), teachers (class resources, grading), TAs (assignment assistance), and students (coursework and grades).
  • Scalable Architecture: Built with Next.js and MongoDB to handle data-driven user experiences and fast rendering.
